{\small\color{heading}\textbf{Conclusion.}}
The requested output is technically possible. The prototype copies every
\texttt{Default} transition to a new class and wraps each run in the current
CJK font. Productizing that mixed class would, however, invalidate xeCJK's
basic CJK/non-CJK class split across many subsystems.
