\section*{Recommended: \texttt{pxrubrica}}

PXrubrica supports XeTeX directly and does not load the traditional
\texttt{CJK.sty}.  The extra braces keep the complete base string in one
group, as in the example previously posted in this issue:

\begin{center}
  \ruby{{四种脱氧核糖核苷酸}}{AGCT}
\end{center}

PXrubrica follows Japanese ruby conventions, so its detailed layout should
not be expected to match the legacy \texttt{ruby.sty} output exactly.
